# Create Dashboard Page

## Configure Login Page

Make sure login page redirect to the dashboard

Make sure after login we save the user data into local storage

```tsx
try {
  const response = await api.post("/auth/login", {
    email,
    password,
  });
  console.log("Login Success:", response.data);

  // Save Token
  localStorage.setItem("accessToken", response.data.token);

  // Save User Info
  localStorage.setItem("user", JSON.stringify(response.data.user));

  // Redirect to dashboard
  window.location.href = "/dashboard"; // simple redirect
} catch (error) {
  const errorMessage =
    error instanceof axios.AxiosError
      ? error.response?.data
      : error instanceof Error
        ? error.message
        : "Unknown error";
  console.error("Login Failed:", errorMessage);
}
```

## Create Dashboard Component

### File: Dashboard.tsx

```tsx
import React, { useEffect, useState } from "react";
import { useNavigate } from "react-router-dom";
import type { User } from "../types/user";

const Dashboard: React.FC = () => {
  const navigate = useNavigate();
  const [user, setUser] = useState<User | null>(null);

  useEffect(() => {
    const storedUser = localStorage.getItem("user");
    if (storedUser) setUser(JSON.parse(storedUser));
  }, []);

```

Explanation:

- useState stores the current user.
- useEffect fetches user data from localStorage when the component loads.
- useNavigate lets you programmatically go to other pages.
