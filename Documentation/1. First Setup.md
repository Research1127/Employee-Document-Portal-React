# Setup The React Project

Run `npm create vite@latest employee-portal-react -- --template react-ts` to create the react project

Choose no for the default question during installation

Then, we run below command:

```shell
cd employee-portal-react
npm install
npm run dev
```

Then run `npm install tailwindcss @tailwindcss/vite `to install tailwind

### Configure the vite plugin

Add the @tailwindcss/vite plugin to your Vite configuration.

```tsx
export default defineConfig({
  plugins: [react(), tailwindcss()],
});
```

---

### Add Tailwind to CSS

Open `src/index.css` and replace everything with:

```tsx
@import "tailwindcss";
```

---
