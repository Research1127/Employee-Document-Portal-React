# Create View Document Page

**OVERVIEW**

This page displays full details of a selected document, including related Department, Category, and Uploader information.

## Routing Configuration

### File: App.tsx

Add route with dynamic id parameter.

```tsx
<Route
    path="/documents/:id"
    element={
      isAuthenticated ? <ViewDocument /> : <Navigate to="/login" replace />
    }
  />
</Routes>
```

```yaml
Part                  | Description
-----------------------------------------------------------------
/documents/:id        | Dynamic route to retrieve document by ID
useParams()           | Used inside ViewDocument to access id
isAuthenticated       | Protects route from unauthorized access
<Navigate />          | Redirects unauthenticated users
```

## Create View Document Page

### File: ViewDocument.tsx

üß† Step 1: Imports & Setup

```tsx
import React, { useEffect, useState } from "react";
import { Link, useNavigate, useParams } from "react-router-dom";
import api from "../api/axios";
import type { Document } from "../types/document";
```

---

üß† Step 2: Component State

```tsx
const { id } = useParams<{ id: string }>();

const [document, setDocument] = useState<Document | null>(null);
const [loading, setLoading] = useState(true);
const [error, setError] = useState<string | null>(null);
const [deleting, setDeleting] = useState(false);

const navigate = useNavigate();
```

---

üß† Step 3: Fetch Document Data

```tsx
useEffect(() => {
  const fetchDocument = async () => {
    try {
      const response = await api.get(`/documents/${id}`);
      setDocument(response.data);
    } catch (err) {
      setError("Failed to fetch document");
    } finally {
      setLoading(false);
    }
  };

  if (id) {
    fetchDocument();
  }
}, [id]);
```

‚úÖ API Used

```yaml
GET /documents/{id}
```

---

üóë Step 4: Delete Function

```tsx
const handleDelete = async () => {
  if (!id) return;

  const confirmDelete = window.confirm(
    "Are you sure you want to delete this document?",
  );

  if (!confirmDelete) return;

  try {
    setDeleting(true);
    await api.delete(`/documents/${id}`);

    alert("Document deleted successfully!");
    navigate("/documents");
  } catch (error) {
    alert("Failed to delete document.");
  } finally {
    setDeleting(false);
  }
};
```

‚úÖ API Used

```yaml
DELETE /documents/{id}
```

## Update document.ts

Need to insert Uploader, Department and DocumentCategory part

### File: types/document.ts

```ts
export interface Document {
  id: number;
  title: string;
  description: string;
  file_name: string;
  file_path: string;
  file_type: string;
  file_size: number;
  category_id: number;
  department_id: number;
  uploaded_by: number;
  access_level: string;
  download_count: number;
  created_at: string;
  updated_at: string;
  department?: Department;
  category?: DocumentCategory;
  uploader?: Uploader;
}

export interface Department {
  id: number;
  name: string;
}

export interface DocumentCategory {
  id: number;
  title: string;
}

export interface Uploader {
  id: number;
  name: string;
  email: string;
}
```

## Backend Requirement (Important ‚ö†Ô∏è)

Your Laravel backend must load relationships:

```php
public function show(Document $document)
{
    return $document->load(['department', 'category', 'uploader']);
}
```

Otherwise:

```tsx
document?.department?.name;
```

will return undefined.
